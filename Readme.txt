*** Python Project ***
Project Name : Virtual Mouse

Description :

It is a Python Project.
It is a Virtual Mouse which recognizises finger gestures, on the movement of index finger and Thumb it will move cursor.
It uses Webcam to capture the hand gestures.

Distance between Thumb and Index finger is measured.
If the measurement is <20 then it is recongnised as a Click event, else it is refered as a movement of cursor on screen.

Technologies Used :
1. OpenCV
2. MediaPipe
3. PyAutoGUI
4. protobuf
